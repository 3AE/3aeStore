一个追梦者的独白

曾经梦想仗剑走天涯
曾经朝远方独行浪子
曾经模仿过很多强者
最后才明白，
无尽的旅程，如此漫长……
想过当一个writer
想过当一个painter
却不料成了loser

最后咬咬牙，

还是打算成为一个coder……

因此才有了下面的故事

虽然不长

只因刚开始……

然而很长

又因无结束……

茫茫人海，繁华世界

再美的故事

不及，此时此刻的不期而遇……
